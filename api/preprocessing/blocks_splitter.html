Splits blocks based on the distribution of buildings within them.

Parameters

blocksgpd.GeoDataFrame

GeoDataFrame containing block data. Must contain the following columns: - index : int - geometry : Polygon

buildingsgpd.GeoDataFrame

GeoDataFrame containing building data. Must contain the following columns: - index : int - geometry : Point

Raises

AssertionError

If the Coordinate Reference Systems (CRS) of blocks and buildings do not match.
